# Chatbot Widget Assignment

This repository contains a frontend chatbot widget built as part of a skill evaluation assignment.
The implementation focuses on clean UI/UX, clear interaction flow, predefined options, and basic user information collection, as required in the assignment.This currently has no backend and the responses have been hardcoded.

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Running the Project
```bash
cd chatbot-widget
npm install
npm run dev
```
